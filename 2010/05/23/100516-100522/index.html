抽籤的公平理論

所謂此

就不管是先抽後抽

抽到 中獎 或是 中了金馬獎 機率都一樣

拿三個白球 和 四個黑球做實驗

第一次拿到白球的機率是 3/7

第2次拿到白球的可能是 白白 黑白

機率是 3/72/6 + 4/73/6等於 3/7

第三次拿到白球的可能是 白白白 黑白白 黑黑白 白黑白

機率是 3/72/61/5 + 4/73/62/5 + 4/73/63/5 + 3/74/62/5 等於 3/7

從此可知 第n次拿到白球的機率也是 3/7

所以 不關先抽後抽 機率都相同

所以 當兵抽籤的時候 應該跟長官說 不用多花時間去抽順序籤
